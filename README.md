# Cronize

### Cronize is a web-app for managing different cronjobs split by project.

## Installation

1.  Clone the repo

    ```ts
        git clone https://github.com/mtlh/cronize.git
    ```

2.  Install dependencies

    ```ts
        npm install
    ```

3.  Run the app

    ```ts
        npm run dev
    ```

4.  View the app at http://localhost:4321
